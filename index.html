Breadth First Search

Breadth First Search (BFS) is a search algorithm used for navigating graph data structures. It utilizes a "breadth-first" strategy, meaning that each node explores each of its neighbor nodes before moving onto the nodes at the next level of depth. It was invented in 1945 by Konrad Zuse.
